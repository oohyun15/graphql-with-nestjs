## Description

GraphQL with NestJS

## Info

- Language: typescript
- Framework: nestjs, [apollo](https://github.com/nestjs/nest/tree/master/sample/23-graphql-code-first)
- Database: mysql
- ORM: [typeorm](https://github.com/nestjs/nest/tree/master/sample/05-sql-typeorm)
- Console: [REPL](https://gist.github.com/royshouvik/2cba46b56dd70529731a068eb82ca538#file-console-ts)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# sync database
$ yarn typeorm schema:sync
```

## Console

```bash
# REPL console
$ npm run console
```

### Example

```node
// user
usersService = app.get('UsersService');
usersService.create({firstName: "Harry", lastName: "Potter"})
usersService.findAll().then((user) => console.log(user))

// kakao webtoon
kakao = app.get('KakaoService);
kakao.create({identifier: '2507'}); // 라지에르의 서, id: 1
kakao.crawl(1);
```
